Hydrous Chatbot Widget

Este es el widget de chatbot para soluciones de reciclaje de agua. A continuación se muestra un ejemplo de cómo integrarlo en tu sitio web:

<!-- Hydrous AI Chatbot Widget -->
<script src="https://[tu-usuario].github.io/hydrous-chatbot-widget/hydrous-widget.min.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    HydrousWidget.init({
      primaryColor: '#0891b2',  // Personalizable
      title: 'Asistente Técnico',
      subtitle: 'Consultas sobre reciclaje de agua'
    });
  });
</script>
